Why a framework?

Speed-up and guide your development

The expertise acquired through hundreds of Oracle APEX projects centralized in a single framework

The goal of APEXFramework is to offer a reusable set of APIs and Plugins to speed-up and guide your developments of Oracle APEX applications by providing proven solutions to common architecture and development issues.

You don't need to reinvent the wheel. This is not rocket science, build up on what others have challenged and solved.

Your time is precious do not waste it.

Why Choose APEXFramework?

Minimize The Uncertainties In Your Projects

Users

Automated user management: Account activation, forgotten password, and credentials retrieval.

Access Rights

A GUI lets you define the granular access to pages and components for the security groups.

Security

Define domains, Create security groups and Add users.

Applications Groups

Dynamic and flexible breadcrumb: Pull values between pages. Stack contextual data based on your navigation.

DTAP cycle

Export applications groups using a few clicks. Import hundreds of applications with their metadata using a single command line.

Notification merge

Create a set of notifications, such as an email that is sent to new users. Each email has the same kind of information, yet the content is unique.
